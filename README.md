# boot-jruby - eval JRuby code from [boot](http://boot-clj.com/).

## Installation

boot-jruby is published to [Clojars](https://clojars.org/boot-jruby). To use from boot:

    [boot-jruby "0.1.0"]

## Usage

See `boot jruby -h` for details.
